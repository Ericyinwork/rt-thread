# git cmd

项目级别仓库用户名设置：

* git config user.name Ericyinwork

* git config user.email ericyinwork@163.com

查看仓库用户名设置

* cat .git/config

系统级别仓库用户名设置：
* git config --global user.name Ericyinwork
* git config --global user.email ericyinwork@163.com

查看系统仓库用户名设置

* cat ~/.gitconfig
